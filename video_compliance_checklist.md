# 视频合规性检查清单

> 团队：热点电商自动化流水线 / 废物 制作
> 日期：2026-03-30

---

## 一、国内平台AI内容合规要求

### 微博
- 2024年起，平台要求AI生成内容必须显著标注
- 禁止利用AI虚假宣传商品（尤其是医疗、金融类）
- 禁止未经授权使用明星/网红形象

### Bilibili
- AI生成视频需在简介/评论区明确标注
- 虚拟主播/AI内容有专门的标签系统
- 禁止AI合成侵权内容

### 小红书
- 要求AI生成内容添加"AI生成"标签
- 带货内容需标明"广告"或"合作"字样
- 禁止虚假宣传

### 抖音/字节系
- AI生成内容必须添加"创作者AI"标识
- 数字人/虚拟主播需额外报备
- 禁止AI换脸侵权内容

## 二、视频合规检查清单（生成前）

### 热点内容审核
- [ ] 热点不涉及政治敏感内容
- [ ] 热点不涉及社会负面事件（灾难/犯罪等）
- [ ] 热点不涉及未成年人相关内容
- [ ] 热点不涉及医疗/金融虚假宣传

### 产品关联审核
- [ ] 产品有合法销售资质
- [ ] 产品宣传语符合《广告法》要求
- [ ] 产品价格/促销信息真实
- [ ] 不涉及特殊品类（药品/医疗器械/保健品）需额外资质

### Prompt审核
- [ ] 不含政治/色情/暴力关键词
- [ ] 不含虚假夸大用语（如"最便宜""绝对有效"）
- [ ] 不含侵犯他人品牌/肖像权的描述

---

## 三、视频生成后检查

### 内容检查
- [ ] 视频画面不包含违禁内容（政治/色情/暴力/赌博）
- [ ] 视频字幕/旁白不包含违规词汇
- [ ] 背景音乐有合法授权或无版权风险
- [ ] 视频时长符合平台要求（通常1分钟以内）

### AI标识嵌入
- [ ] 视频左上角/右下角有"AI生成"水印
- [ ] 水印清晰可见，不影响主要内容
- [ ] 发布平台的标题/简介中有AI标注

### 发布前复核
- [ ] 人工复核视频内容
- [ ] 确认产品信息准确
- [ ] 确认合规标识已嵌入
- [ ] 确认目标平台已申请必要权限

## 四、电商带货特殊合规

### 广告法合规
```
禁止用语清单：
- 最低价/全网最低/极致性价比
- 绝对/100%/完全保证
- 药用功效描述（除非有资质）
- 虚假销量/好评数据
```

### 特殊品类限制
| 品类 | 额外要求 |
|---|---|
| 食品 | 需食品流通许可证，包装信息完整 |
| 化妆品 | 需备案号，功效宣称有依据 |
| 药品/医疗器械 | 禁止AI自动生成推广内容 |
| 金融产品 | 禁止AI自动生成，需专业审核 |

### 价格合规
- 促销价格必须真实（不得先涨后降）
- 折扣比例需有依据
- 限时促销需明确截止时间

## 五、合规审核流程

```
热点抓取 → LLM初步筛选 → 可疑标记 
    ↓
人工复核（可疑热点） → 通过/拒绝
    ↓
Prompt构造 → Prompt审核 → 通过/修改
    ↓
视频生成 → AI自动检查 → 可疑标记
    ↓
人工复核（可疑视频） → 通过/重新生成
    ↓
FFmpeg处理 → AI标识嵌入 → 最终复核
    ↓
发布
```

---

## 六、违规应对机制

### 发现违规内容已发布
1. 立即下架/删除内容
2. 记录违规原因
3. 更新审核规则
4. 如需主动报备（政治类）立即上报

### 平台处罚应对
- 轻度违规（警告）→ 整改后申诉
- 重度违规（限流/封禁）→ 分析原因，整改后申诉
- 永久封禁 → 启用备用账号，调整策略
