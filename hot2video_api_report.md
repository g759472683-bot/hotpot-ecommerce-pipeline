## 二、第一阶段推荐方案

**结论：采用 MiniMax Hailuo 作为主力 + 快手可灵作为备选**

### 推荐理由：
1. **MiniMax Hailuo**
   - 已完成实测验证，可用 ✅
   - 中文Prompt理解能力优秀
   - API文档完善，有Python示例
   - 价格合理（10秒视频约¥2-3）

2. **快手可灵（备选）**
   - 10秒支持好，1080P稳定
   - 短视频平台，电商场景天然契合
   - 有免费试用额度

### 不采用原因：
- **腾讯混元**：需企业账号，个人开发者门槛高
- **通义万相**：文档为英文，集成成本高
- **Runway/Pika**：国外服务，国内访问不稳定，支付麻烦

## 五、FFmpeg模板化方案（第一阶段过渡）

在AI视频质量不稳定期间，使用FFmpeg生成模板化视频作为保底：

```bash
# 示例：图片+字幕+背景音乐 → 视频
ffmpeg -loop 1 -i image.jpg -i audio.mp3 \
  -vf "drawtext=text='AI生成内容':fontsize=24:fontcolor=white:x=10:y=10" \
  -t 10 -c:v libx264 -c:a aac output.mp4
```

**优点**：成本为零，稳定可控
**缺点**：缺乏AI生成的创意性和灵活性

**策略**：前期模板为主，AI生成为辅；后期逐步提高AI占比。
